# ⚡ Sysbench CPU Benchmark Guide (Linux / VPS)

Ein einfaches Tutorial, um die CPU-Leistung eines Linux VPS mit sysbench zu testen.

------------------------------------------------------------

📦 Installation

sudo apt update
sudo apt install sysbench neofetch -y

------------------------------------------------------------

🧠 Was ist Sysbench?

sysbench ist ein leichtgewichtiges Benchmark-Tool für:

- CPU
- RAM
- Disk I/O
- Datenbanken

In diesem Guide fokussieren wir uns auf CPU Performance.

------------------------------------------------------------

🚀 1. Single-Core Benchmark

sysbench cpu run

→ Testet nur 1 CPU-Thread (≈ 1 Core)

Beispiel:
CPU speed:
    events per second: 5579.33

→ Score = events per second

------------------------------------------------------------

⚡ 2. Multi-Core Benchmark

sysbench --threads=$(nproc) cpu run

→ Nutzt alle verfügbaren CPU-Kerne

------------------------------------------------------------

📊 Wichtige Werte

events per second:
- Haupt-Score
- Je höher → desto schneller

latency:
- Reaktionszeit der CPU
- Niedriger = besser

total number of events:
- Gesamt berechnete Aufgaben

------------------------------------------------------------

🧪 Beispiel Vergleich

VPS A → 1200   (schwach)
VPS B → 5500   (gut)
VPS C → 30000  (sehr stark)

------------------------------------------------------------

🧠 Einflussfaktoren

- CPU Architektur (wichtigster Faktor)
- IPC (Instructions per Cycle)
- Takt (GHz)
- Anzahl Kerne
- Cache-Größe
- VPS Anbieter (Shared vs Dedicated)

------------------------------------------------------------

⚠️ Hinweise

- VPS kann gedrosselt oder geteilt sein
- Mehr Kerne ≠ automatisch mehr Leistung
- Moderne CPUs sind oft effizienter

------------------------------------------------------------

🔍 CPU Infos anzeigen

lscpu

oder

cat /proc/cpuinfo | grep "model name" | head -1

------------------------------------------------------------

💡 Tipps

✔ Single + Multi-Core testen
✔ Anbieter vergleichen
✔ Auf moderne CPU achten

------------------------------------------------------------

🧾 Fazit

- sysbench ist ideal für schnelle CPU-Tests
- events/sec = Benchmark Score
- Moderne CPUs sind deutlich schneller

------------------------------------------------------------

⭐ Bonus

sysbench cpu --threads=$(nproc) --time=30 run

------------------------------------------------------------

Happy Benchmarking 🚀
